# GPT Learning

A collection of experiments and implementations built while learning how GPT-style language models work from first principles.

## What this repository covers

- Tokenization
- Attention mechanism
- Context windows
- Transformer architecture
- Training a small GPT-style model
- Experiments with PyTorch

## Project Structure

```
.
├── experiments/      # Standalone experiments
├── modular_gpt/      # Modular GPT implementation
├── notebooks/        # Jupyter notebooks and exploration
└── torch/            # PyTorch learning exercises
```

## Technologies

- Python
- PyTorch
- Jupyter Notebook
- Weights & Biases (for experiment tracking)

## Purpose

This repository documents my journey of understanding large language models by implementing core concepts instead of only using existing libraries.

It serves as a learning log rather than a production-ready framework.
