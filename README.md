# GPT Learning

A collection of experiments and implementations built while learning how GPT-style language models work from first principles.

Rather than relying solely on high-level libraries, this repository focuses on understanding the core components that power modern Large Language Models (LLMs) by implementing and experimenting with them from scratch.

---

## Topics Covered

- Tokenization
- Self-Attention
- Context Windows
- Transformer Architecture
- Training a Small GPT-style Model
- PyTorch Experiments
